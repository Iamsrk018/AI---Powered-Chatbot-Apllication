# AI---Powered-Chatbot-Apllication
An advanced AI-powered chatbot built using React, Tailwind CSS, Vite, and Google Gemini API. It delivers real-time, context-aware conversations through a clean and responsive interface. Designed for speed, simplicity, and scalability — perfect for integrating smart conversational AI into modern web applications.

##  Features
- Smart conversations using Gemini API  
- Clean UI with Tailwind  
- Fast builds with Vite  

##  Tech Stack
React • Vite • Tailwind CSS • Google Gemini API
